## 기능 목록

- Header 구현

  - 장바구니에 담긴 상품 전체의 갯수 구현

- 상품 목록 구현

  - 상품 목록 레이아웃 구현 (grid)
  - mock 데이터를 기반으로 정렬하기

- 상품 아이템 구현

  - img 파일 import 하기

- 장바구니 버튼
  - 장바구니 버튼을 클릭하면 상품이 1개가 담긴다.
  - 장바구니 버튼이 숫자 인풋으로 바뀐다.
  - 장바구니에 담을 상품의 갯수를 조절할 수 있다.
